## ğŸ¯ Common Tasks

### As a Customer

**Place Order**
1. Browse menu
2. Add to cart
3. Customize items
4. Checkout
5. Pay

**Schedule Order**
1. Choose "Schedule for later"
2. Pick date and time
3. Restaurant must accept within advance window
4. Receive confirmation

**Track Order**
1. Click order number in confirmation email
2. View real-time status
3. See estimated ready time
4. Chat with restaurant if needed

### As Kitchen Staff

**View Orders**
- New orders appear automatically
- Audio alert plays
- Kitchen ticket auto-prints (if configured)

**Manage Order**
1. Review order details
2. Accept or reject (if time unavailable)
3. Update status: Preparing â†’ Ready
4. Customer gets notified automatically

**Handle Rush**
- Orders queue automatically
- Sort by priority (ASAP vs scheduled)
- Update ETAs if needed

### As Admin

**Add Menu Item**
1. Navigate to Menu
2. Click "Add Item"
3. Upload photo
4. Set price and description
5. Add modifiers (sizes, extras)
6. Assign to printer station
7. Publish

**Configure Delivery**
1. Enable delivery in settings
2. Create delivery zones
3. Set fees per zone
4. Define minimum orders
5. Test with sample address

**View Reports**
1. Dashboard shows today's summary
2. Filter by date range
3. Export to CSV
4. View popular items
5. Analyze pickup vs delivery
